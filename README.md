# zotero笔记主题

## 介绍

zotero本身的note样式比较朴素,在编辑器里提供了用于自定义CSS的设置就催生了这个库.

欢迎大家访问下载

## 网页

[在线下载网页](https://zottheme.wk8686.top/)

## 使用方法

1. 打开zotero软件，进**入编辑-设置-高级界面**
2. 打开**编辑器**
3. 搜索CSS，在extensions.zotero.note.css项直接填入CSS代码即可
